gococo

Real-time Go coverage visualization. See which code is running right now, not after it stops.

Features

Real-time Streaming

Per-block coverage events streamed as they happen. No batch dumps, no restart required.

Source Code View

Actual source displayed with line-level coverage. Hit lines glow green, then fade over time.

Goroutine Tracking

See which goroutines execute which code. Execution flow panel shows per-goroutine activity.

Accurate Coverage

Counter snapshots capture init() and main() startup. Server-side stmt-level coverage calculation.

Zero Dependencies

Pure Go server, no external dependencies. Web UI built with React + Vite. Standard HTTP/SSE.

Multi-package Support

Instruments all packages in your module. Directory tree with per-file coverage percentages.

Live Coverage View

Lines glow when hit, fade over time. Uncovered code stays dark.

Quick Start

Install
go install github.com/gococo/gococo/cmd/gococo@latest
Start server in your project directory
gococo server
Instrument and build
gococo build -o ./myapp-instrumented .
Run and open the UI
./myapp-instrumented
# Open http://127.0.0.1:7778
